# Mục tiêu: 
Bạn là một chuyên gia Frontend đặc biệt là Reactjs, hãy đóng vài trò là mentor của tôi đễ hướng dẫn giúp tôi build dự án , đặc biệt chia sẽ công nghệ mới nhất sử dụng mục đích cho đi làm. Tôi có kinh nghiệm nhiều năm làm Angular
Chúng ta sẽ build dự án về ứng dụng tiếng anh
# Kĩ thuật:
- Tôi sẽ sử dụng frontend-workspace để làm việc với Nx command
- Next.js 15 (App Router)     ← Framework chính
- TypeScript                  ← Bắt buộc, bạn đã quen từ Angular
- TanStack Query v5           ← Data fetching 
- Zustand                     ← State management
- React Hook Form + Zod       ← Form (quen với Angular Forms sẽ dễ)

# Cấu trúc
packages: Dùng để build core các thư viện dùng typescripts đóng gói, theo chuẩn mới nhất, có thể publish
apps: Chứa application, đây tôi sẽ sử dụng Next.js

Hiện tại tôi đã build front-workspace
Hãy add project Next.js vào frontend-workspace/apps giúp tôi
Với việc clear file READ.md, hướng dẫn các cách setup để scartch project và updated

- Hướng dẫn cách setup thành công project dựa vào Nx
